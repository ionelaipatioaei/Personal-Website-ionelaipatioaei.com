Skills

HMTL5 & CSS3

These are the building blocks for every website, thus it is important to write HTML & CSS which is easy to understand and maintain. Using CSS flexbox, grid and front-end libraries like Bootstrap and Materialize correctly will help your website look good and make the code easier to read.
If you need a unique design for your website though, writing clean HTML & CSS is one of my strong skills.

Modern Javascript

Javascript defines the way a website behaves so it represents an important building block for your website. I usually use the latest Javascript features - ES6 and up - but if needed I can produce Javascript which will run even on Internet Explorer.
I can either use vanilla JS to manipulate th DOM, jQuery or more advanced libraries like React depending on your needs.

React & React Router

React is one of the most popular front-end framework and makes building interactive UIs painless, thus making webapp building and maintaining easier. I mainly used React to build my TSXYZ project.
React Router is a free library which helps you to handle client-side routing. The routing on the TSXYZ webapp was done using React Router.

NodeJS & MongoDB

NodeJS is Javascript ran on the server side. It takes care of all back-end logic like loging, apis, routing, etc. Usually NodeJS is used in combination with ExpressJS.
MongoDB is a noSQL database which is popular to use with NodeJS. If your website needs to log user information then you'll need a database. I usually pair MongoDB with Mongoose.

Python

Python is a general-purpose language. You can do almost everything with it, from neural networks to websites. If you need a simple script for whatever reason then Python is the best choice. I mainly use it for my non-web personal projects or web scrapping.
Some of the libraries which I used are P5PY and Pyglet for my Nature of Code project. I aslo have an Youtube series where I mainly use Python.

I'm a Fast Learner

I can adapt to use new technologies pretty quickly. New framework or library which you want me to use? No problem I can figure out on my own the basics.
One of the best examples is how I learned Python. In around one week I was able to write basic Python code and use it for my projects. While I was by no means an expert at writing good Python code, I knew enough to get things done.
